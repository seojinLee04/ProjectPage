Abstract

건강검진 데이터를 사용하여 혈압을 예측하는 것은 건강 관리와 질병 예방에 중요한 역할을 합니다. 혈압은 심혈관 질환의 주요 위험 요소 중 하나로, 이를 정확히 예측하고 관리함으로써 개인의 건강을 유지하고 질병 발생 가능성을 줄일 수 있습니다.
로지스틱 회귀(Logistic Regression) 모델을 사용하고 교차 검증을 통해 모델의 성능을 평가하고, 평균 F1-score를 계산하여 모델의 예측 정확도를 확인합니다.

Method

주어진 건강검진 데이터에서 모든 데이터가 결측치인 부분을 삭제하고, 고혈압이 아닌 사람들 중에서 고혈압 환자 수만큼 무작위로 선택하여 언더 샘플링(Undersampling)을 한 후 BMI를 추가하여 전처리를 하였습니다.
로지스틱 회귀(Logistic Regression) 모델을 사용하여 고혈압 예측 모델을 만들었고, 하이퍼파라미터(Hyperparameter)는 (penalty='l1', fit_intercept=True, solver='saga', random_state=1, multi_class='multinomial', max_iter=100)로 설정하였습니다.

Experimental Results

실험 결과, 모델의 정확도는 약 83%였으며, F1 스코어는 0.00으로 나타났습니다. 이는 모델이 고혈압 여부를 예측하는 데 있어 어려움을 겪고 있음을 의미합니다. 데이터 불균형 문제를 해결하기 위해 클래스 가중치를 적용했지만, 여전히 모델의 성능을 개선하기 위한 추가적인 조치가 필요함을 시사합니다.
